住宿安排：

两晚蒙古利亚大酒店,其位于新老城区交界处的锡林湖湖心岛，道路畅通。西邻蒙元文化博物馆，南邻锡林浩特市植物园，北邻锡林浩特市图书馆，环境优美，空气清新。酒店内外设计装修致力于传承蒙元文化、独具匠心。一梁一柱，一帛一画，无不包含蒙古元素，尽显蒙元气质。酒店服务设施先进，功能齐全，是锡林浩特市最 大的集住宿、餐饮、会议、文化礼仪为一体的蒙元文化主题酒店。

餐饮特色：

诈马宴是蒙古族特有的庆典宴飨整牛席或整羊席。诈马，蒙语是指退掉毛的整畜，意思是把牛、羊家畜宰杀后，用热水退毛，去掉内脏，烤制或煮制上席。

行程亮点：
锡林郭勒千里草原风景大道

该线路途径御马苑、贡宝拉格草原、石条山、浑善达克沙地，再抵达“中国马都”锡林浩特，随后往东北经东乌珠穆沁旗抵达电影《狼图腾》故事发生地、外景拍摄地—乌拉盖，每一站都堪称草原游牧之魂。

贝子庙祭敖包

贝子庙，蒙古语名“班智达葛根庙”，汉名“崇善寺”。是内蒙古四大庙宇之一，始建于清乾隆八年（1743年），历经七代活佛精修而成，贝子庙是藏传佛教圣地。它的规模，历来被称为锡林郭勒盟地区第一大寺，是锡林郭勒盟佛教文化的一大宝库，享有“北国名刹”声誉。祭敖包是蒙古民族隆重的祭祀活动，也是蒙古族最为隆重热烈的祭祀活动，活动当天前往贝子庙祭敖包，邀请喇嘛祈福，祈祷国泰民安、风调雨顺、出行平安。

民族歌舞晚会/篝火晚会、民俗表演、盛装马术
感受草原激情与豪放

根据地大乐队参与过中国数十次音乐节，是中国深圳的一支老牌摇滚乐队，连续举办过十一届音乐节，被誉为文化风向标、最具有情怀的音乐节。

乌兰牧骑，蒙语原意为“红色的嫩芽”，意为红色文化工作队，活跃于农村牧区间。1957年诞生在内蒙古自治区锡林郭勒盟苏尼特右旗。始终坚持不懈地全心全意为农牧民服务，被农牧民亲切地称为“玛奈（我们的）乌兰牧骑”，乌兰牧骑队员则被唤作“玛奈呼和德（我们的孩子）”。

中国在中蒙边境1046号界碑处，历史上称为“蒙马处”，是中蒙两国边民世代友好的通商要道和重要枢纽。本次活动，将首次组织重机车队护送“环中国边境骑行接力车”前往界碑、国门献礼互动。

闭幕式

中蒙摩旅合作开启，为所有参加本次活动的骑士颁发珠恩嘎达布其镇“荣誉居民”称号。见证历史时刻，奏响重机新征程。

跨境摩旅体验

跨境摩旅体验周将组织20台越野拉力摩托车队进入蒙古国骑行，国内的摩托车爱好者将与蒙古国的车手交流、互动。一路上大漠戈壁，蓝天白云，牛羊点点，秀美的蒙古风光，纯正的蒙古国美食，深厚的蒙古国文化，都将展现在你眼前。
