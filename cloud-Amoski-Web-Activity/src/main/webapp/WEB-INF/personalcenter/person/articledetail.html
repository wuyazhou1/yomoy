爱摩老斯基的柳州之行，“愚你有礼”让你爆

更新于21小时前

2015年第一届环中国，上百位骑士分别位于祖国边境多个点， 同时顺时针出发，在最短的时间内利用GPS轨迹完成首次中国版图骑行；

2016年第二届环中国，俱乐部成员再次顺时针完成环边境；

2017年第三届环中国，以传递接力棒的方式，数百名万里骑士及支持者参与了此次活动， 并留下了非常珍贵的“接力棒”及签字长幅；同年12月，万里骑士062号阿诺， 单人单骑无后援完成环中国边境连续骑行活动，历时127天总里程达38835公里；
